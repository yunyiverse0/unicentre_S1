# 1. 기본 입출력 및 자료형

## input()
- 입력은 기본적으로 문자열(str)로 들어온다.
- 여러 값 입력 시 `split()` 사용.

## 자료형 종류
- int: 정수
- float: 실수
- str: 문자열
- bool: True / False

## 문자열 특징
- immutable (수정 불가능)
- 슬라이싱 가능
- 작은따옴표/큰따옴표 모두 사용 가능

## 숫자 진법
- 0b: 2진수  
- 0o: 8진수  
- 0x: 16진수

# 3. 조건문

## 기본 구조
```
if 조건:
    수행문
elif 조건:
    수행문
else:
    수행문
```

## 특징
- 들여쓰기(indent)로 블록 구분
- 조건식은 bool 결과를 가져야 함
- 중첩 사용 가능

# 4. 반복문 (for, while)

## for 문
- 특정 횟수 반복에 사용
- range()와 함께 자주 사용

## range()
- range(stop)
- range(start, stop)
- range(start, stop, step)

## while 문
- 조건이 참일 동안 계속 반복
- 탈출 조건 필수

## break / continue
- break: 반복 종료
- continue: 다음 반복으로 건너뜀

# 6. 자료형 1: 리스트 / 튜플 / 딕셔너리 / 세트

---

# 6-1. 리스트(list)

## 특징
- mutable (값 변경 가능)
- 인덱싱 가능
- 슬라이싱 가능
- 다양한 내장 메서드 제공

## 주요 성질
- 순서가 있다
- 중복 가능
- 수정 가능

---

# 6-2. 튜플(tuple)

## 특징
- immutable (값 변경 불가능)
- 리스트와 구조는 동일하지만 수정 불가
- 괄호 ( ) 사용
- 수정 불가능하므로 안정적 자료 보관용

## 성질
- 순서가 있다
- 중복 가능
- 수정 불가능

---

# 6-3. 딕셔너리(dictionary)

## 특징
- key:value 형태
- key는 중복 불가
- 값은 중복 가능
- 순서가 유지됨(Python 3.7+)

## 주요 용도
- 구조화된 데이터 저장
- 빠른 검색

---

# 6-4. 세트(set)

## 특징
- 중복 없음
- 순서 없음
- 집합 연산 지원 (합집합, 교집합, 차집합)

## 주요 사용처
- 중복 제거
- 집합 연산

---

# 8. 정렬 및 lambda

## sorted()
- 새로운 정렬된 리스트 반환
- 원본 유지

## list.sort()
- 리스트를 제자리에서 정렬
- 원본 변경

## 정렬 옵션
- reverse: True → 내림차순
- key: 정렬 기준 함수

## lambda 개념
- 이름 없는 함수
- 정렬 기준으로 자주 사용됨

예: key=lambda x: x[0] (개념만)
