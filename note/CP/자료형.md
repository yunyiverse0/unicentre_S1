# [ 원소 추가 및 수정 ]

## < list > (순서O, 수정O)

### 추가
1. `append(value)` : 맨 마지막에 value 추가
2. `insert(index, value)` : 원하는 위치에 value 추가
3. `extend([v1, v2])` : 여러 개의 value 추가

### 수정
- 인덱스로 바로 수정 가능  
  ```python
  lst[index] = new_value
  ```

### 삭제
1. 값 삭제: `lst.remove(value)`
2. 인덱스로 삭제: `lst.pop(index)` / `del lst[index]`
3. 여러 개 삭제: `del lst[1:4]`  (index 1~3 삭제)
4. 전체 삭제: `lst.clear()`

---

## < set > (순서X, 수정O)

### 추가
1. `set.add(value)` : 값 하나 추가
2. `set.update([v1, v2])` : 여러 개의 값 추가

### 집합 연산

#### 1. 합집합 (Union)
- `a | b`
- `a.union(b)`
- `a |= b` (in-place)

#### 2. 교집합 (Intersection)
- `a & b`
- `a.intersection(b)`
- `a &= b` (in-place)

#### 3. 차집합 (Difference)
- `a - b`
- `a.difference(b)`
- `a -= b` (in-place)

### 삭제
1. 값 삭제  
   - `set.remove(x)` : 값 없으면 오류  
   - `set.discard(x)` : 값 없어도 통과  
2. 아무거나 하나 삭제: `set.pop()`
3. 여러 개 삭제  
   - `s -= {1, 2, 3}`  
   - `s.difference_update({1, 2, 3})`
4. 전체 삭제: `set.clear()`

---

## < dictionary > (key:value)

### 추가
1. 새로운 key 추가: `d['a'] = 10`
2. 여러 개 추가: `d.update({'b': 30, 'c': 40})`

### 수정
- value 수정: `d['a'] = 20`

### 삭제
1. key 삭제: `del d['a']` / `d.pop('a')`
2. 전체 삭제: `d.clear()`

---

## < tuple >

### 추가 / 수정
튜플은 immutable → 직접 수정 불가능.

#### 수정하려면:
1. tuple → list 변환
2. 값 추가/수정
3. 다시 tuple로 변환

```python
t = (1, 2, 3)
lst = list(t)
lst.append(4)
t = tuple(lst)
```
