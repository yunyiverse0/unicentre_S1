# 09. 함수 (Function)

## 함수의 의미
- 특정 작업을 수행하는 코드 블록
- 코드 재사용, 유지보수가 쉬움
- 프로그램 구조화에 용이

---

## 함수 구조
```
def 함수이름(매개변수):
    문장
    return 값
```

---

## 기본 예시
### 1) 매개변수와 반환값이 있는 함수
```python
def add(a, b):
    return a + b
```

### 2) 반환값이 없는 함수
```python
def greet(name):
    print("Hello,", name)
```

### 3) 매개변수 기본값(default)
```python
def area(width, height=10):
    return width * height
```

---

## 매개변수 / 인수
- parameter: 함수 선언 시 변수
- argument: 호출 시 전달하는 실제 값

---

## return
- 함수 실행 종료
- 값을 호출한 곳에 반환
- return이 없으면 None 반환

---

## 변수 스코프 (Scope)

### 지역 변수(local)
```python
def f():
    x = 10   # 지역 변수
```

### 전역 변수(global)
```python
x = 10
def f():
    global x
    x = 20
```

---

## 키워드 인수 (keyword argument)
```python
print(name="Yun", age=20)
```

---

## 가변 매개변수
### *args (여러 개의 위치 인수)
```python
def total(*nums):
    return sum(nums)
```

### **kwargs (여러 개의 키워드 인수)
```python
def info(**data):
    return data
```

---

## 함수의 종류 정리
- 매개변수 O / 반환값 O  
- 매개변수 O / 반환값 X  
- 매개변수 X / 반환값 O  
- 매개변수 X / 반환값 X

# 10. 예외 처리 & 파일 처리

# 예외 처리(Exception Handling)

## 개념
- 프로그램 실행 중 발생하는 오류를 처리하는 구조
- 강제 종료 방지

## try ~ except 기본 구조
```
try:
    실행 코드
except 오류타입:
    예외 처리
```

## 문법 요소
- try: 예외 발생 가능성 있는 코드
- except: 예외 발생 시 실행되는 코드
- else: 예외가 없을 때 실행
- finally: 예외 여부 관계없이 항상 실행

---

# 파일 처리(File Handling)

## 파일 열기 모드
- r : 읽기
- w : 쓰기(기존 삭제)
- a : 추가
- b : 바이너리

## 파일 처리 순서
1. open()
2. read/write
3. close()

## with 문
- 파일 객체를 자동으로 닫아줌
```
with open("a.txt", "r") as f:
    ...
```

# 11. 클래스 & 객체 (OOP 기본)

## 클래스(Class)
- 객체를 생성하기 위한 설계도

## 객체(Object)
- 속성(변수) + 동작(메서드)으로 이루어진 실체

## OOP 특징
- 캡슐화
- 상속
- 다형성
- 추상화

---

## 클래스 구조
```
class ClassName:
    def __init__(self):
        ...
    def method(self):
        ...
```

## self
- 객체 자기 자신을 가리킴

## 생성자(Constructor)
- 객체 생성 시 자동 호출: `__init__`

## 인스턴스 변수
- `self.변수명` 형태로 선언하여 객체마다 독립 존재

## 접근자 / 설정자
- getter / setter 역할
- 데이터 보호 목적

## is-a 관계

- 자동차는 차량이다
